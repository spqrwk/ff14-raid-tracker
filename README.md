# 🎯 FF14 高难副本开荒记录

> 专为固定队设计的犯错追踪与开荒进度管理工具，纯浏览器运行，无需后端。

---

## 📖 这是什么

固定队开荒时，每个人都会犯错——记不住谁犯了什么错、哪把最远到哪、今天效率如何？这个工具帮你**记录每一把的犯错和进度**，自动生成图表统计，让问题一目了然。

数据全部保存在**浏览器本地**，不上传任何服务器。换个浏览器或清缓存会丢，记得定期导出备份。

## 📝 第三步：开荒记录

回到 **「开荒面板」**，这里是日常使用的核心页面。

### 记录犯错

```
☑ MT 张三  ☐ ST 李四  ☑ H1 王五  ☐ H2 ...
☐ D1 ...   ☐ D2      ☐ D3      ☐ D4

P几：  P3         ← 选或输入阶段
内容：吃机制了      ← 选填

犯错等级：
○ 减员        ○ 团灭        ● 狂暴        ○ 罪无可恕
  不影响继续     本把结束      输出不足        重大失误
```

- **勾选**犯错的队员（可多选）
- 选择阶段、填写内容（选填）
- 选择犯错等级
- 点提交

> 💡 **团灭、狂暴、罪无可恕** 会自动结束本把，下一把 Pull 号 +1

### 记录进度

到达某个阶段但没有犯错？切到「记录进度」标签：

```
到达P几：P5
备注：    顺利处理了三运
☐ 记录后结束本把

[记录进度]
```

### 手动结束 / 通关

- **手动结束**：这把自己灭了但没记录到，点它结束
- **🏆 本把通关**：过了！自动标记「已完成」并结束

## 🔍 FFLogs 查询（v1.2+）

无需导入数据，直接在线查询 FFLogs 上的开荒记录。需要 FFLogs API 凭据（免费注册）。

### 进度统计

查某个玩家在指定 Boss 上的所有战斗记录，自动去重，生成：

- **卡点分布**柱状图 — 哪个阶段死最多
- **阶段通过次数**图 — P1/P2/P3…各有多少把通过
- **进度趋势**散点图 — 进步了还是在坐牢
- **详细表格** — 每把的阶段、血量%、是否通关、可搜索筛选导出 JSON/CSV

### 初通队友对比

查某个玩家的首次通关记录，列出 7 个队友，统计每人此前的通关次数：

| 列 | 含义 |
|---|---|
| 此前 | 该队友早于目标初通的通关次数 |
| 同场 | 与目标同一时间窗口的通关次数 |
| 此后 | 晚于目标初通的通关次数 |
| 当前 | 该队友当前总通关次数 |

同时还展示目标玩家自己的各职业通关次数和最高 rDPS。

> 💡 队友查询会自动解析角色 ID，优先用高效接口一次获取全部记录，避免翻页浪费 API 额度。

## 💾 数据安全

**「数据管理」** 页面提供：

| 操作 | 用途 |
|------|------|
| 导出全部 | 完整备份，JSON 文件下载 |
| 导出队员 | 仅备份队员名单 |
| 完整导入 | 从备份文件恢复全部数据 |
| 仅导入队员 | 只导入队员名单，不影响记录 |
| 阶段顺序 | 自定义每个副本的阶段（P1~P7 等） |
| 清空数据 | 输入确认文字 + 弹窗确认，双重保险 |

> ⚠️ **重要**：定期导出备份！清除浏览器缓存会导致数据丢失。

## ❓ 常见问题

**Q：换电脑了数据怎么迁移？**

旧电脑上「数据管理」→「导出全部」→ 把 JSON 文件传到新电脑 → 「完整导入」。

**Q：一个角色位有多个替补怎么加？**

「成员管理」批量表格里，点角色行右侧的 `+` 新增一行。

**Q：无犯通关怎么记录？**

点开荒面板的「🏆 本把通关」按钮，自动标记完成并结束本把。

**Q：多人同时犯错，选了团灭会变成好几把？**

不会。多人勾选后提交，所有人在**同一把**里记录，团灭只结束一次。
